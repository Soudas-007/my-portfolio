# my-portfolio

This is my high-end, pixel-art inspired portfolio website built with Next.js, Framer Motion, and Tailwind CSS.
It features immersive cinematic interactions, a custom cursor, and retro-modern design aesthetics.

## Features
- Immersive 3D Tilt Cards
- Magnetic Hero Buttons
- Custom Particle Trail Cursor
- Parallax Night-mode Contact Section
- Pixel-Art Creative Toolbox
